# S3 Static Website - AWS

## Project Overview
This project demonstrates hosting a static website on AWS S3.  
It covers creating an S3 bucket, uploading files, configuring static website hosting, and troubleshooting public access issues.

## Files
- `index.html` → Homepage of the website
- `screenshots/` → Evidence of bucket setup and website live

## Steps Taken
1. Created an S3 bucket in AWS.
2. Uploaded `index.html` and optional CSS/images.
3. Enabled static website hosting in bucket properties.
4. Configured bucket policy to allow public access.
5. Tested the live website using the endpoint URL.
6. Captured screenshots for documentation.
